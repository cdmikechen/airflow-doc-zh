## 流程

### 一、认领

首先查看[整体进度](https://github.com/apachecn/airflow-doc-zh/issues/1)，确认没有人认领了你想认领的章节。
 
然后回复 ISSUE 来认领章节，注明“章节 + QQ 号”。

### 二、校对

校对：

+   语法
+   术语使用
+   代码格式

### 三、提交

+   `fork` Github 项目
+   修改`zh`下面的文档
+   `push`
+   `pull request`

请见 [Github 入门指南](https://github.com/apachecn/kaggle/blob/dev/docs/GitHub)。
